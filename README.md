
# How JavaScript Actually Works

A practical deep dive into the concepts that power JavaScript under the hood.

This book focuses on the mental models behind JavaScript rather than memorizing syntax.

Topics include:

* Stack vs Heap Memory
* References and Memory Allocation
* Shallow Copy vs Deep Copy
* Execution Context
* Hoisting
* Temporal Dead Zone
* Scope and Closures
* Prototype Chain
* The `this` Keyword
* Event Loop
* Promises
* Async/Await
* Type Coercion
* Garbage Collection

The goal is to help developers understand not just what JavaScript does, but why it behaves the way it does.
